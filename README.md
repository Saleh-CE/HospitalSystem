#  Hospital Management System

A C++ application designed to manage hospital operations including doctors, patients, and clinic assignments using Object-Oriented Programming (OOP) principles.

## Features

* Add new doctors with personal and professional details
* Add new patients with medical and personal information
* Assign patients to doctors
* Display doctor details along with assigned patients
* Display patient information
* Assign doctors to specific clinics
* Interactive menu-driven system

## Data Structures Used

* Static Arrays (for storing doctors and patients)

## Technologies

* C++
* Object-Oriented Programming (OOP)
* Standard Library (`iostream`, `string`)

## File Structure

* main.cpp — program entry point
* proj.h — class definitions and function prototypes
* proj.cpp — implementation of classes and system logic
